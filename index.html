The OSI (Open Systems Interconnection) Model is a conceptual framework used to understand and implement interoperable network protocols. It is divided into 7 layers, each with specific functions.

üß± The 7 Layers of the OSI Model

7. Application Layer: Interfaces directly with user applications (e.g., browsers, email).
This layer interacts with software applications and provides network services to end-user processes.
6. Presentation Layer: Translates data formats (e.g., encryption, compression).
It formats or translates data for the application layer, including encryption and compression.
5. Session Layer: Manages sessions between devices (start/end communication).
Establishes, manages, and terminates sessions between communicating hosts.
4. Transport Layer: Ensures reliable transmission (TCP/UDP protocols).
Provides end-to-end communication control and error checking.
3. Network Layer: Handles IP addressing and routing (e.g., IP, routers).
Determines how data is sent to the receiving device via logical addressing.
2. Data Link Layer: Responsible for MAC addressing and frame transmission.
Ensures error-free transfer of data frames between nodes over the physical layer.
1. Physical Layer: Transmits raw bits over physical media (cables, switches).
Deals with the physical connection to the network and the transmission of bits.
